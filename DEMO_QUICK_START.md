Demo Quick Start Guide

STEP 1: Get Google Gemini API Key (Free)

1. Go to: https://aistudio.google.com
2. Sign in with your Google account
3. Click "Get API Key" button
4. Click "Create API Key" 
5. Copy the API key (looks like: AIzaSy...)

Time: 2 minutes

STEP 2: Run Demo

Open terminal/command prompt:

cd C:\Users\user\Desktop\NutriMedAI
python -m streamlit run app_fast_demo.py

STEP 3: Use Demo

1. Browser automatically opens at localhost:8501
2. Select "Google Gemini Vision"
3. Paste your API key
4. Upload food image
5. Click "Analyze Food"
6. Get results in 5-10 seconds!
